راهنمای راه‌اندازی

Outline (گزینه ساده)

برای به‌کارگیری Outline نخست آنرا با یکی از گزینه های زیر دریافت کنید.

Android iOS Windows macOS

نوشته زیر را Copy کنید.

برنامه Outline را باز کنید. بر روی + یا Add server بزنید. نوشته را در جایگاه access key بگذارید. بر روی دکمه ADD SERVER که پس از زمان کوتاهی پدیدار می‌شود بزنید تا Server به برنامه افزوده شود.

VMESS (گزینه پیشنهادی)

برای به‌کارگیری VMESS نخست یکی از برنامه های زیر را دریافت کنید.

Android (v2rayNG) iOS (ShadowLink) Windows (Qv2ray) macOS (V2RayX)

نوشته زیر را Copy کنید.

در برنامه Android:
روی + و سپس Import config from Clipboard بزنید تا Server به برنامه افزوده شود.

در برنامه iOS:
روی Server، سپس + و سپس Server links بزنید تا Server به برنامه افزوده شود.

در برنامه Windows:
روی Import+ بزنید و نوشته را در بخش Share Link بچسبانید و روی Import بزنید.

در برنامه macOS:
روی Configure، سپس ...Import و سپس ...Import from other links بزنید.
